Task 2: Write a Javascript program

From this ZIP file, locate and open Lab5Template.html. Rename the file to Lab5.html
In Lab5.html, write the JavaScript code that

  • Prompts a user to enter a temperature in Celsius - hint you will need to use the prompt() Method
  • Calculates the input temperature in Fahrenheit.
  • Displays an alert with the original temperature, and the converted temperature - hint you will need to use the alert() Method to display your answer variable
